# Food Order App - Backend

Backend API for Food Order Application built with Node.js, Express, and MongoDB.

## Environment Variables

Create a `.env` file with:
```
MONGODB_URI=your_mongodb_atlas_connection_string
PORT=3000
```

## Installation

```bash
npm install
```

## Seed Database

```bash
npm run seed
```

## Run Server

```bash
npm start
```

## API Endpoints

- `GET /meals` - Get all meals
- `POST /orders` - Create new order
